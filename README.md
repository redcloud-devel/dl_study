# 딥러닝 스터디

이 폴더는 딥러닝 기초부터 단계별로 학습하기 위한 노트와 예제 코드를 정리합니다.

## 학습 로드맵 (요약)
- 단계 0: 준비 — 환경 세팅, 로드맵 확정
- 단계 1: 수학 기초 — 선형대수/미분
- 단계 2: 머신러닝 기초 — 지도학습/손실/과적합/정규화
- 단계 3: 딥러닝 기초 — 퍼셉트론/MLP/역전파
- 단계 4: 파이토치 실전 — 데이터/모델/학습루프
- 단계 5: 대표 모델 — CNN 기본
- 단계 6: 미니 프로젝트 — end-to-end 실습

## 진행 현황
- 1주차: `week1/` 예제 코드 + 주석 완료
- 2주차: `week2/` 예제 코드 + 주석/시각화 완료
